Xhamia API

REST API for mosque locations across Albania — 869 records.

Overview

This API provides structured data about mosques across Albania, including name, city, address, and GPS coordinates. All requests require an API key passed in the request header.

Authentication

Every request must include your API key in the x-api-key header. Requests without a valid key return 401 Unauthorized.

GET /api/mosques
x-api-key: YOUR_SECRET_KEY
⚠️ Never expose your API key in client-side JavaScript. Always route requests through your own backend or use environment variables.

Base URL

BASEhttps://your-project.vercel.app

Replace your-project with your Vercel project name after deploying.

List all mosques

GET/api/mosques

Returns a paginated list of all mosques. Defaults to 50 results per page.

Example request

curl https://your-project.vercel.app/api/mosques \
  -H "x-api-key: YOUR_SECRET_KEY"

Example response

{
  "success": true,
  "meta": {
    "total": 869,
    "page": 1,
    "limit": 50,
    "totalPages": 18
  },
  "data": [
    {
      "id": "001",
      "name": "Xhamia Ahmetaq",
      "city": "Ahmetaq, Tiranë",
      "address": "Rruge fshati, qender",
      "lat": 41.433916,
      "lng": 19.682254
    },
    ...
  ]
}

Get mosque by ID

GET/api/mosques?id=050

Returns a single mosque matching the given id.

Parameters

ParameterTypeDescription
idstringThe mosque ID (e.g. 001, 743)

Example request

curl "https://your-project.vercel.app/api/mosques?id=743" \
  -H "x-api-key: YOUR_SECRET_KEY"

Example response

{
  "success": true,
  "data": {
    "id": 743,
    "name": "Xhamia Ethem Bej",
    "city": "Tiranë",
    "address": "Sheshi Skenderbej",
    "lat": 41.327793,
    "lng": 19.819346
  }
}

Filter by city

GET/api/mosques?city=Tiranë

Returns mosques in the specified city. The match is case-insensitive and partial — searching tiran will match Tiranë.

Example request

curl "https://your-project.vercel.app/api/mosques?city=Shkodër" \
  -H "x-api-key: YOUR_SECRET_KEY"

Filter by name

GET/api/mosques?name=Ethem

Returns mosques whose name contains the given string. Case-insensitive partial match.

Example request

curl "https://your-project.vercel.app/api/mosques?name=Plumbit" \
  -H "x-api-key: YOUR_SECRET_KEY"

Pagination

All list endpoints support pagination via page and limit query parameters.

ParameterDefaultMaxDescription
page1—Page number
limit50100Results per page

Example

GET /api/mosques?city=Tiranë&page=2&limit=20

Response format

All responses are JSON. Successful responses include "success": true. Error responses include "success": false and an "error" message.

Error codes

StatusMeaning
200OK — request successful
401Unauthorized — missing or invalid x-api-key
404Not found — no mosque matches the given ID
405Method not allowed — only GET is supported
